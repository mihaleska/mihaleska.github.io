Code uilder

Code Builder je eno od ključnih stvari v Minecraft Education Edition, če se pogovarjamo o programiranju, ki učencu omogoča učenje programiranja skozi (najpogosteje) premikanje blokov. Podpira pa tudi Python ali JavaScript, zato je primeren tako za učence v osnovni šoli kot tudi za programerje, ki so kar vešči v drugih programskih jezikih.

Laik bi si predstavljal, da se programiranje v Code builderju začne in konča z agentom. Temu namreč ni tako. S Code builderjem se lahko "igramo" z veliko več kot samo z nekim možicem, ki sam po sebi ne dela ničesar. Omogoča spreminjanje časa in vremena v igri, lahko postavljamo pogoje z if / else stavki, izvajamo zanke, izjavno logiko (and, or in podobno), računamo vsote, korene, absolutne vrednosti in tako dalje..
